A better compressed bitset

Roaring bitmaps are compressed bitmaps. They can be hundreds of times faster.

Grab one of our research papers Roaring Bitmaps on GitHub

Widely used

Roaring is found in

  • Apache Lucene and derivative systems such as Solr and Elasticsearch,
  • Apache Druid (incubating),
  • Apache Spark,
  • Apache Hive,
  • Apache Tez,
  • Apache CarbonData,
  • Netflix Atlas,
  • LinkedIn Pinot,
  • OpenSearchServer,
  • Cloud Torrent,
  • Whoosh,
  • InfluxDB,
  • Pilosa,
  • Bleve,
  • Microsoft Visual Studio Team Services (VSTS),
  • Intel’s Optimized Analytics Package (OAP),
  • Tablesaw,
  • Jive Miru,
  • Gaffer,
  • Apache Hivemall,
  • eBay’s Apache Kylin.

Portable

There are freely available software libraries providing Roaring bitmaps in almost all the popular programming languages:

  • Java,
  • C, C++,
  • Go,
  • C#,
  • Swift,
  • Rust,
  • Python,
  • Node/JavaScript,
  • D,
  • …

There is a serialized format specification for interoperability between implementations.
